# 投资者问答库

> 按融资轮次分类的投资者常见问题与优秀回答指南

---

## 轮次差异概览

| 轮次 | 投资人关注重点 | 问题难度 | 回答侧重点 |
|------|----------------|----------|-------------|
| **种子轮** | 团队、方向、热情 | 简单-中等 | 为什么是你们 |
| **A轮** | 增长、数据、单位经济 | 中等-困难 | 为什么现在能成 |
| **B轮** | 规模化、盈利、组织 | 困难 | 为什么能持续领先 |

---

## 一、种子轮问题 (Seed)

### 1.1 团队与创始人

**Q1: 为什么是你们能做这个？**
- **考察点**: 创始人背景、执行能力
- **难度**: ⭐⭐

**优秀回答模板**:
```
创始人A: [背景] → [相关经验]
创始人B: [背景] → [相关经验]

为什么我们能做成:
1. [第一个差异化优势]
2. [第二个差异化优势]
```

**优秀案例**:
> "我曾在[知名公司]负责[核心业务]，亲手从0做到[$X] ARR。我的联合创始人曾是[竞品公司]的核心工程师。"

---

**Q2: 你们全职创业吗？**
- **考察点**: 创始人commitment
- **难度**: ⭐

**优秀回答**:
```
✅ 全部全职，无后顾之忧
✅ 已辞职，全力以赴
🟡 部分全职，正准备全职
```

> 如果不是全职，说明你的机会成本，展示你已经做出了选择。

---

**Q3: 创始人之间是如何分工的？**
- **考察点**: 团队结构、决策机制
- **难度**: ⭐⭐

**优秀回答**:
```
技术/产品: [创始人A]
商业/销售: [创始人B]
运营/客服: [创始人C]

决策机制:
- 技术决策: CTO最终决定
- 商业决策: CEO最终决定
- 争议场景: 投票或外部顾问
```

---

**Q4: 团队还缺什么关键角色？**
- **考察点**: 自知之明、招聘能力
- **难度**: ⭐⭐

**优秀回答**:
```
✅ 我们意识到[岗位]是短板
✅ 正在招聘中，已有人选
✅ 预计[时间]到位

已有资源:
- [招聘渠道]
- [潜在候选人]
```

---

### 1.2 问题与解决方案

**Q5: 你们在解决什么问题？**
- **考察点**: 问题真实性和重要性
- **难度**: ⭐⭐

**优秀回答模板**:
```
问题: [具体描述用户痛点]
影响: [影响范围和程度]
现状: [用户当前如何解决]
代价: [不解决的后果]
```

**优秀案例**:
> "设计师每天花[X]小时手动标注图片，这些工作完全可以自动化。我们让设计团队效率提升[X]倍。"

---

**Q6: 为什么这个问题现在值得解决？**
- **考察点**: 时机判断
- **难度**: ⭐⭐⭐

```

1. [市场因素变化]
2. [技术成熟度提升]
3. [用户习惯改变]
4. [政策/监管影响]

我们恰好在正确的时间出现。
```

---

**Q7: 你们和竞品有什么不同？**
- **考察点**: 差异化、竞争力
- **难度**: ⭐⭐⭐

**优秀回答**:
```
[竞品A]: [做法] → [局限性]
[竞品B]: [做法] → [局限性]

我们: [独特做法] → [独特优势]

一句话定位:
"[竞品]做X时，我们做Y，让用户得到Z。"
```

### 1.3 市场与机会

**Q8: 市场有多大？**
- **考察点**: 市场认知、TAM合理性
- **难度**: ⭐⭐

**优秀回答**:
```
TAM: $500B (全球市场，理论最大)
SAM: $50B (可服务市场，有购买力)
SOM: $2B (可获得市场，第一年)

假设:
- 中国有[X]万家企业
- 其中[X]%有需求
- 每年付费$XXX
- = $X亿可获得市场
```

> ⚠️ 避免: "万亿级市场" (已被投资人听烂)

**Q9: 你的前10个客户在哪里？**
- **考察点**: 可执行力、接地气程度
- **难度**: ⭐⭐⭐

**优秀回答**:
```
目标客户画像: [描述]
已接触: [数量]家
意向明确: [数量]家

具体名单(可选):
- [客户A]: [需求描述]
- [客户B]: [需求描述]

```

---

**Q10: 如果失败了，原因会是什么？**
- **考察点**: 风险意识、诚实度
- **难度**: ⭐⭐⭐

**优秀回答**:
```
主要风险:
1. [风险1] → 应对措施
2. [风险2] → 应对措施

我们的判断: 最大的风险是[风险]，但我们[应对方案]。
```

> ⚠️ 别说"不会失败" - 投资人想看到你有风险意识
