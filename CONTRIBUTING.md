# 贡献指南

## 如何贡献

### 案例贡献
欢迎提交新的创业失败/成功案例：
1. 必须是真实可查的案例
2. 包含详细的失败节点分析
3. 提取可复用的反模式

### Skill 改进
- 优化分析流程
- 添加新的分析维度
- 完善风险评估模型

### 文档完善
- 修正错误信息
- 补充案例细节
- 改善可读性

## 提交规范

### 案例格式
```markdown
## [公司名] (年-年)

### 基本信息
- 赛道：
- 阶段：
- 融资情况：

### 失败时间线
1. [时间] 关键事件

### 失败模式
- 模式1
- 模式2

### 教训提炼
- 核心教训
```

### PR 要求
- 清晰的标题
- 案例来源标注
- 至少一个提取的反模式
